# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0-alpha.11] - 2026-03-19

### Documentation
- Updated `Quickstart` to explicitly label **Reown Project ID** as a mandatory requirement for wallet connectivity.
- Added a new **Prerequisites** section to the Introduction page.
- Added high-visibility warning boxes for mandatory wallet configuration.

### Templates
- Updated basic template `main.tsx` and `.env` to simplify project ID configuration.

## [2.0.0-alpha.10] - 2026-03-16
- Initial v2.0 alpha release.
- Added `FanTokenGate`, `PredictionCard`, `WagerCard`, and `PollCard`.
- Integrated Chiliz Chain support.
